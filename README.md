## Quick Start

WILL ONLY RUN IF ALL MICROSOFT EDGE PROGRAMS CLOSED, BACKGROUND TASKS COUNT

## 1. Clone the Repository
Clone the project to your local machine.

## 2. Create and Activate a Virtual Environment
Create a virtual environment and ensure `pip` is available.

```bash
python -m venv venv
source venv/bin/activate        # macOS/Linux
venv\Scripts\activate           # Windows
```

modify user data file location, profile if needed, and webdriver executable location

python main.py
